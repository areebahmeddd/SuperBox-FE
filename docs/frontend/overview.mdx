## Architecture Principles

### Component-Based Design

SuperBox follows a modular component architecture:

```tsx
src/
├── components/          # Reusable UI components
│   ├── auth-modal.tsx
│   ├── server-card.tsx
│   ├── header.tsx
│   └── ...
├── app/                 # Next.js App Router pages
│   ├── layout.tsx
│   ├── page.tsx
│   └── [routes]/
└── lib/                 # Shared utilities and types
    └── types.ts
```

### State Management

<Info>
  SuperBox uses React 19's built-in state management with Server Components for
  optimal data fetching and minimal client-side JavaScript.
</Info>
