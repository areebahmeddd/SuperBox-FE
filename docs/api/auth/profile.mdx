```

```

## Update Profile

### Endpoint

```
PATCH /auth/profile
```

### Request Body

<ParamField body="username" type="string">
  New username
</ParamField>

<ParamField body="bio" type="string">
  User biography (max 500 characters)
</ParamField>

<ParamField body="website" type="string">
  Website URL
</ParamField>

<ParamField body="location" type="string">
  User location
</ParamField>

<ParamField body="avatar_url" type="string">
  Avatar image URL
</ParamField>

### Example Request

<CodeGroup>
```bash cURL
curl -X PATCH https://api.superbox.com/v1/auth/profile \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "bio": "Building MCP servers for AI",
    "website": "https://johndoe.dev",
    "location": "San Francisco"
  }'
```

```javascript JavaScript
const response = await fetch("https://api.superbox.com/v1/auth/profile", {
  method: "PATCH",
  headers: {
    Authorization: `Bearer ${token}`,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    bio: "Building MCP servers for AI",
    website: "https://johndoe.dev",
    location: "San Francisco",
  }),
});
```

```python Python
response = requests.patch(
    'https://api.superbox.com/v1/auth/profile',
    headers={'Authorization': f'Bearer {token}'},
    json={
        'bio': 'Building MCP servers for AI',
        'website': 'https://johndoe.dev',
        'location': 'San Francisco',
    }
)
```

</CodeGroup>

### Response (200)

```json
{
  "id": "usr_1234567890",
  "username": "johndoe",
  "bio": "Building MCP servers for AI",
  "website": "https://johndoe.dev",
  "location": "San Francisco",
  "updated_at": "2024-01-15T10:35:00Z"
}
```

## Change Password

### Endpoint

```
POST /auth/change-password
```

### Request Body

<ParamField body="current_password" type="string" required>
  Current password
</ParamField>

<ParamField body="new_password" type="string" required>
  New password (min 8 characters)
</ParamField>

### Example Request

<CodeGroup>
```bash cURL
curl -X POST https://api.superbox.com/v1/auth/change-password \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "current_password": "OldPass123!",
    "new_password": "NewSecurePass456!"
  }'
```

```python Python
response = requests.post(
    'https://api.superbox.com/v1/auth/change-password',
    headers={'Authorization': f'Bearer {token}'},
    json={
        'current_password': 'OldPass123!',
        'new_password': 'NewSecurePass456!',
    }
)
```

</CodeGroup>

### Response (200)

```json
{
  "message": "Password changed successfully"
}
```

## Delete Account

### Endpoint

```
DELETE /auth/profile
```

### Request Body

<ParamField body="password" type="string" required>
  Current password for confirmation
</ParamField>

<ParamField body="confirm" type="boolean" required>
  Must be `true` to confirm deletion
</ParamField>

### Example Request

```bash
curl -X DELETE https://api.superbox.com/v1/auth/profile \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{
    "password": "SecurePass123!",
    "confirm": true
  }'
```

### Response (200)

```json
{
  "message": "Account deleted successfully"
}
```

<Warning>
  Account deletion is permanent and cannot be undone. All servers and data will
  be deleted.
</Warning>
