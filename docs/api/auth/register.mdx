---

## Endpoint

```
POST /auth/register
```

## Request Body

<ParamField body="email" type="string" required>
  User email address
</ParamField>

<ParamField body="username" type="string" required>
  Unique username (3-50 characters)
</ParamField>

<ParamField body="password" type="string" required>
  Password (min 8 characters)
</ParamField>

## Example Request

<CodeGroup>
```bash cURL
curl -X POST https://api.superbox.com/v1/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "email": "john@example.com",
    "username": "johndoe",
    "password": "SecurePass123!"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.superbox.com/v1/auth/register', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    email: 'john@example.com',
    username: 'johndoe',
    password: 'SecurePass123!',
  }),
});

const data = await response.json();
```

```python Python
import requests

response = requests.post(
    'https://api.superbox.com/v1/auth/register',
    json={
        'email': 'john@example.com',
        'username': 'johndoe',
        'password': 'SecurePass123!',
    }
)

data = response.json()
```
</CodeGroup>

## Response

<ResponseField name="user" type="object">
  User object
  
  <Expandable title="properties">
    <ResponseField name="id" type="string">
      User ID
    </ResponseField>
    
    <ResponseField name="email" type="string">
      User email
    </ResponseField>
    
    <ResponseField name="username" type="string">
      Username
    </ResponseField>
    
    <ResponseField name="created_at" type="string">
      Account creation timestamp
    </ResponseField>
  </Expandable>
</ResponseField>

<ResponseField name="token" type="string">
  JWT authentication token
</ResponseField>

<ResponseField name="refresh_token" type="string">
  Refresh token for obtaining new access tokens
</ResponseField>

### Error Responses

<CodeGroup>
```json 400 - Validation Error
{
  "error": "Validation failed",
  "details": {
    "email": "Invalid email format",
    "password": "Password must be at least 8 characters"
  }
}
```

```json 409 - Conflict
{
  "error": "User already exists",
  "message": "Email or username is already registered"
}
```

```json 429 - Rate Limit
{
  "error": "Rate limit exceeded",
  "message": "Too many registration attempts. Try again in 1 hour."
}
```
</CodeGroup>
