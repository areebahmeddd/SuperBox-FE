## Endpoint

<Card>
  <code>GET https://api.superbox.dev/api/v1/servers</code>
</Card>

## Authentication

<Tabs>
  <Tab title="Optional">
    This endpoint works with or without authentication.
    <Info>
      Authenticated requests have higher rate limits (1000/hour vs 100/hour)
    </Info>
  </Tab>
</Tabs>

## Query Parameters

<ParamField query="page" type="integer" default="1">
  Page number for pagination (1-indexed)
</ParamField>

<ParamField query="limit" type="integer" default="20">
  Number of servers per page (max: 100)
</ParamField>

<ParamField query="lang" type="string">
  Filter by programming language **Allowed values:** `python`, `javascript`,
  `typescript`, `go`, `rust`
</ParamField>

<ParamField query="author" type="string">
  Filter by server author/creator
</ParamField>

<ParamField query="pricing" type="string">
  Filter by pricing model **Allowed values:** `free`, `paid`
</ParamField>

<ParamField query="search" type="string">
  Search servers by name or description
</ParamField>

<ParamField query="sort" type="string" default="created_at">
  Sort field and order **Examples:** - `name` - Sort by name (A-Z) - `-name` -
  Sort by name (Z-A) - `-downloads` - Most downloads first - `-created_at` -
  Newest first - `pricing.amount` - Lowest price first
</ParamField>

<ParamField query="security_passed" type="boolean">
  Filter by security scan status **Values:** - `true` - Only servers that passed
  security scans - `false` - Servers with security issues
</ParamField>

## Examples

<CodeGroup>
```bash
cURL
# List all servers (default pagination)
curl -X GET "https://api.superbox.dev/api/v1/servers" \
  -H "Content-Type: application/json"

# List Python servers only

curl -X GET "https://api.superbox.dev/api/v1/servers?lang=python" \
 -H "Content-Type: application/json"

# List free servers with security passed

curl -X GET "https://api.superbox.dev/api/v1/servers?pricing=free&security_passed=true" \
 -H "Content-Type: application/json"

# Search and sort

curl -X GET "https://api.superbox.dev/api/v1/servers?search=weather&sort=-downloads" \
 -H "Content-Type: application/json"

# With authentication

curl -X GET "https://api.superbox.dev/api/v1/servers?page=2&limit=50" \
 -H "Authorization: Bearer $SUPERBOX_API_TOKEN" \
 -H "Content-Type: application/json"
````

```javascript JavaScript
// Basic request
const response = await fetch('https://api.superbox.dev/api/v1/servers');
const data = await response.json();
console.log(data);

// With filters
const filters = new URLSearchParams({
  lang: 'python',
  pricing: 'free',
  security_passed: 'true',
  page: '1',
  limit: '20'
});

const response = await fetch(
  `https://api.superbox.dev/api/v1/servers?${filters}`

  }
});

// Process results
data.data.forEach(server => {
  console.log(`${server.name} v${server.version} - ${server.description}`);
});
````

```python Python
import requests

# Basic request
url = "https://api.superbox.dev/api/v1/servers"
response = requests.get(url)
data = response.json()

# With filters
params = {
    'lang': 'python',
    'pricing': 'free',
    'security_passed': True,
    'page': 1,
    'limit': 20,
    'sort': '-downloads'
}

response = requests.get(url, params=params)
data = response.json()

# With authentication
headers = {
    'Authorization': f'Bearer {os.getenv("SUPERBOX_API_TOKEN")}',
    'Content-Type': 'application/json'
}

response = requests.get(url, headers=headers, params=params)
data = response.json()

# Process results
for server in data['data']:
    print(f"{server['name']} v{server['version']} - {server['description']}")
```

```go Go
package main

import (
    "encoding/json"
    "fmt"
    "net/http"
    "net/url"
)

func main() {
    // Build URL with query parameters
    baseURL := "https://api.superbox.dev/api/v1/servers"
    params := url.Values{}
    params.Add("lang", "python")
    params.Add("pricing", "free")
    params.Add("page", "1")
    params.Add("limit", "20")

    fullURL := fmt.Sprintf("%s?%s", baseURL, params.Encode())

    // Create request
    req, err := http.NewRequest("GET", fullURL, nil)
    if err != nil {
        panic(err)
    }

    // Add authentication (optional)
    token := os.Getenv("SUPERBOX_API_TOKEN")
    if token != "" {
        req.Header.Set("Authorization", "Bearer "+token)
    }

    // Make request
    client := &http.Client{}
    resp, err := client.Do(req)
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    // Parse response
    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)

    fmt.Printf("Found %v servers\n", len(result["data"].([]interface{})))
}
```

    def get_all_servers():
        all_servers = []
        page = 1
        
        while True:
            response = requests.get(
                'https://api.superbox.dev/api/v1/servers',
                params={'page': page, 'limit': 100}
            )
            data = response.json()
            
            all_servers.extend(data['data'])
            
            if not data['meta']['pagination']['hasNext']:
                break
            
            page += 1
        
        return all_servers

## Best Practices

<CardGroup cols={2}>
  <Card title="Use Pagination" icon="page">
    Always implement pagination for large result sets to improve performance
  </Card>

<Card title="Cache Results" icon="database">
  Cache server lists on the client side with appropriate TTL (5-10 minutes)
</Card>

<Card title="Filter Server-Side" icon="filter">
  Use query parameters instead of filtering on the client to reduce data
  transfer
</Card>

  <Card title="Handle Rate Limits" icon="gauge">
    Implement exponential backoff when rate limits are hit
  </Card>
</CardGroup>

<Tip>
  For better performance, use authenticated requests which have higher rate
  limits and may return cached results faster.
</Tip>
