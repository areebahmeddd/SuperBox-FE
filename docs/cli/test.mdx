## Usage

```bash
superbox test [OPTIONS]
```

## Description

Executes the test suite for your MCP server with coverage reporting.

## Options

<ParamField path="--coverage" type="boolean">
  Generate coverage report
  
  ```bash
  superbox test --coverage
  ```
</ParamField>

<ParamField path="--verbose" type="boolean">
  Detailed test output
</ParamField>

<ParamField path="--pattern" type="string" default="test_*.py">
  Test file pattern
</ParamField>

<ParamField path="--failfast" type="boolean">
  Stop on first failure
</ParamField>

## Examples

<CodeGroup>
```bash Basic Test
superbox test
```

```bash With Coverage
superbox test --coverage --verbose
```

```bash Specific Pattern
superbox test --pattern "integration_*.py"
```
</CodeGroup>
