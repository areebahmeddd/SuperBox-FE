## Usage

```bash
superbox run [OPTIONS]
```

## Description

Starts local MCP server for testing tools interactively before publishing.

## Options

<ParamField path="--port" type="number" default={8080}>
  Server port ```bash
superbox run --port 3000
```
</ParamField>

<ParamField path="--debug" type="boolean">
  Enable debug logging
</ParamField>

<ParamField path="--watch" type="boolean">
  Auto-reload on file changes
</ParamField>

## Examples

<CodeGroup>
```bash
Basic Run
superbox run
# Server running on http://localhost:8080
```

```bash
With Debug
superbox run --debug --watch
```

```bash
Custom Port
superbox run --port 5000
```

</CodeGroup>
