## Usage

```bash
superbox inspect [SERVER_ID] [OPTIONS]
```

## Description

Display comprehensive information about an MCP server including tools, metadata, and statistics.

## Arguments

<ParamField path="SERVER_ID" type="string" required>
  Server ID to inspect
  
  ```bash
  superbox inspect weather-mcp-123
  ```
</ParamField>

## Options

<ParamField path="--tools" type="boolean">
  Show only tools list
</ParamField>

<ParamField path="--json" type="boolean">
  Output as JSON
</ParamField>

## Examples

<CodeGroup>
```bash
Basic Inspect
superbox inspect weather-mcp-123
```

```bash
Tools Only
superbox inspect weather-mcp-123 --tools
```

```bash
JSON Output
superbox inspect weather-mcp-123 --json > server.json
```
</CodeGroup>
